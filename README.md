# ecommerce-funnel-analytics

Небольшой pet-проект по продуктовой аналитике e-commerce продукта.

Я сделал его, чтобы потренироваться в анализе пользовательских событий, расчёте продуктовых метрик, построении воронки и проверке простой A/B-гипотезы.

## Что внутри

В проекте есть синтетический датасет с событиями пользователей интернет-магазина:

```text
session_start -> view_item -> add_to_cart -> begin_checkout -> purchase
```

Каждая строка в датасете — это одно событие пользователя.

Основные поля:

- `user_id` — пользователь;
- `session_id` — сессия;
- `event_timestamp` — время события;
- `event_name` — тип события;
- `traffic_source` — источник трафика;
- `device_category` — устройство;
- `product_category` — категория товара;
- `revenue` — выручка;
- `ab_group` — группа A/B-теста.

Данные синтетические, поэтому в них нет реальных персональных данных.

## Что я считаю

В проекте рассчитываются:

- продуктовая воронка;
- конверсия между шагами;
- общая конверсия в покупку;
- DAU;
- revenue;
- AOV;
- ARPU;
- ARPPU;
- retention;
- конверсия по источникам трафика;
- A/B-тест для конверсии `view_item -> add_to_cart`.

## Дашборд

После запуска скрипта создаётся интерактивный HTML-дашборд на Plotly.

В нём есть:

- график воронки;
- динамика DAU;
- конверсия по источникам трафика;
- retention;
- сравнение групп A и B.

Файл дашборда:

```text
dashboard.html
```

Его можно просто открыть в браузере.

## Как запустить

Сначала установить зависимости:

```bash
pip install -r requirements.txt
```

Потом запустить основной скрипт:

```bash
python src/product_metrics.py --input data/events.csv --output reports --charts_dir charts --dashboard dashboard.html
```

После этого появятся:

- графики в папке `charts/`;
- таблицы с метриками в папке `reports/`;
- интерактивный дашборд `dashboard.html`.

## Что получилось

На текущих синтетических данных проект считает основные метрики e-commerce продукта и показывает, где пользователи теряются в воронке.

Также есть простая проверка A/B-гипотезы: сравнивается конверсия из просмотра товара в добавление в корзину для групп A и B.
